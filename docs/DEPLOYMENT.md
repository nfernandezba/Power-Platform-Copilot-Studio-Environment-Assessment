# Deployment

1. Upload the complete contents of this package to the repository root.
2. Keep the supplied relative folder structure unchanged.
3. Configure GitHub Pages as `Deploy from a branch` → `main` → `/(root)`.
4. Wait for the Pages deployment to complete and perform a hard refresh.

The assessment has no authentication dependency and processes answers locally in the browser.

The packaged sample report is available at `assets/reports/Power_Platform_Copilot_Studio_Environment_Strategy_Assessment_Sample_Report_v1.0.pdf`.
